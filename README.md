# Tic Tac Toe CLI

A command-line Tic Tac Toe game built with TypeScript and Commander.

## Installation

```bash
npm install
```

## Building

```bash
npm run build
```

## Playing the Game

```bash
npm start
```

Or use the CLI directly:

```bash
node dist/index.js play
```

## How to Play

- The board is numbered 1-9 (displayed when game starts)
- Players take turns entering a number to place their mark (X or O)
- First player to get three in a row wins
- If all positions are filled with no winner, it's a draw

## Development

- **Build**: `npm run build`
- **Run**: `npm start` or `npm run dev`
